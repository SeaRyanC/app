## LLL - Little League Lineup

A game scheduling tool for Little League baseball coaches. Enter your roster, mark who's present, and configure which positions each player is eligible for — then generate a fair, randomized lineup for the whole game with one click.

The scheduler randomizes the player order (which doubles as the batting order), then fills each inning using a round-robin system: no player repeats a position until everyone else eligible has played it at least once. Infield slots (P, C, 1B, 2B, 3B, SS) are filled first, so players who can cover constrained roles naturally earn fewer Off innings without any special configuration. The single "OF" position holds up to three players per inning — coaches decide the exact outfield alignment in real time. The optimizer runs for up to 200 ms, generating as many candidates as possible and selecting the one that best minimizes consecutive high-intensity (infield) and low-intensity (OF) innings for each player. The lineup table shows each player's batting-order number alongside their IF, OF, and Off inning counts; a companion "By Position" view shows who is at each spot every inning. Use the Print button for a printable two-page landscape PDF, or the Share button to send a link encoding the full lineup. Your roster and settings are saved automatically so you're always ready for the next game.
