## LLL - Little League Lineup

A game scheduling tool for Little League baseball coaches. Enter your roster, mark who's present, and configure which positions each player is eligible for — then generate a fair, randomized lineup for the whole game with one click.

The scheduler distributes playing time equitably using a round-robin system: no player repeats a position until everyone else eligible has played it (including time off). The optimizer runs for up to 400 ms, generating as many candidate lineups as possible and selecting the best one — prioritizing evenly-spaced bench innings, Off time adjacent to high-intensity (infield) stints, and dedicated rest before and after pitching or catching appearances. The generated lineup table shows each player's infield (IF), outfield (OF), and Off inning counts at a glance, and a second "By Position" table lets you quickly see who is at each spot every inning. Use the Print button to generate a printable two-page landscape PDF, or the Share button to copy a link encoding the full lineup for easy sharing with parents or assistants. Your roster and settings are saved automatically so you're always ready for the next game.
