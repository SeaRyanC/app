## Label Mail Merge

A browser-based tool for creating printable labels through mail merge. Paste CSV or TSV data, design your label template with placeholders, select from 30+ paper formats including Avery templates, and generate a professional PDF ready for printing.

Key features include automatic header detection, Markdown formatting support (**bold**, *italic*, ~~underline~~), configurable alignment and padding, optional cut marks and border lines, and persistent state via localStorage so you never lose your work.
