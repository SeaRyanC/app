## WSG - Word Search Generator

Generate printable word search puzzles for kids ages 5–10. Choose from 20 themed
word pools—animals, space, food, fantasy, and more—or pick "Random" to draw from
all of them. Set the number of words, choose how many puzzles you want, and
preview a live grid before downloading a multi-page PDF with answer keys, sized
for US Letter or A4 paper.
